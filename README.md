# Weather App 🌦️

A simple weather application built with **HTML, CSS, and JavaScript** that fetches real-time weather data using the [OpenWeather API](https://openweathermap.org/api).

## ✨ Features
- Search weather by **city name**
- Fetch weather using **current location (GPS)**
- Displays:
  - City & Country
  - Temperature (°C)
  - Feels Like
  - Humidity
  - Wind Speed
  - Weather Condition
- Clean, modern UI design
